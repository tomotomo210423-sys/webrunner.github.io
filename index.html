パーフェクト使い方ガイド 📖 (Ver.1.0)

Web File Runner へようこそ！🎉

このツールは、あなたのスマホやPCに入っているプログラムのファイルを、
インターネット上のサーバーを使わずに、ブラウザだけで安全に動かせる魔法のツールです✨
アプリを作りたい！自作ゲームをテストしたい！そんな時に大活躍します。

基本の使い方と「魔法の合体機能」🧩

「ファイルを開く」ボタンからファイルを選んで、「コードを実行」を押すだけでOK！
さらに、複数のファイルを同時に選ぶと、ツールがファイルの種類を自動で推理し、一つのWebアプリとして合体させてから動かしてくれます。

ファイルの役割（Webアプリの仕組み）

  • 📄 .html (骨組み): 家でいうと「柱」や「壁」。画面に表示される文字やボタンの配置を決めます。
  • 🎨 .css (装飾): 家でいうと「壁紙」や「家具」。色を変えたり、オシャレなデザインに整えたりします。
  • ⚙️ .js (動作): 家でいうと「電気」や「水道」。ボタンを押した時にキャラが動いたり、敵にダメージを与えたりする「仕組み」を作ります。（JavaScriptと呼ばれます）

💡 豆知識: 例えば「index.html」「style.css」「game.js」の3つを一緒に選んで実行すると、ツールが自動で「HTMLの骨組みに、CSSの服を着せて、JSの筋肉を繋げる」というプロの裏技をやってくれます！

開発を助ける強力な機能 🚀

  • 🔍 スマホでも見れる「コンソール」: プログラムに間違い（エラー）があったり、裏側の計算結果を見たい時（console.logと言います）、画面の下に黒い画面がスッと出てきて教えてくれます。スマホでプログラミングの勉強をする時にめちゃくちゃ便利です！
  • 💾 完成品の「保存（ダウンロード）」: エディタで書き換えたり、合体させたプログラムを、「1つの完成したHTMLファイル」としてあなたの端末に保存できます。作ったゲームを友達に配ることもできちゃいます。
  • ⚡ サクサク開発「自動実行」: 画面右上のスイッチをONにすると、文字を打つたびに勝手にプレビューが更新されます。色やサイズを細かく調整する時にストレスゼロです。
  • 📝 ブラウザだけでプログラミング開始: エディタのタブの右側にある「＋」ボタンを押せば、新しいカラのファイルを作れます。思いついたらすぐにここから作り始められます！

よくある質問・困ったときは（FAQ）❓

🖼️ ゲームで使う画像やBGMは入れられるの？
はい、入れられます！
.png や .jpg、.mp3 などの素材ファイルも一緒に選んでアップロードしてください。コードの中に <img src="hero.png"> のようにファイル名を書くだけで、ツールが自動で結びつけて表示してくれます。
💾 RPGなどのセーブデータは消えない？
ブラウザの「ローカルストレージ（安全な金庫のような場所）」に保存されます。ブラウザの履歴やキャッシュ（ゴミ箱のようなもの）を空にしない限り、データはちゃんと残るので安心してください。
📱 スマホだとプレビュー画面が小さくて遊びにくい…
プレビュー画面の右上にある「全画面で表示」ボタンを押してみてください！邪魔な枠が消えて、スマホの画面いっぱいにゲームが広がるので最高に遊びやすくなります。
📄 「.txt」っていうただのメモ帳のファイルでも動く？
動きます！
ツールが中身のコードを読んで「あ、これはHTMLだな」「これはJavaScriptのプログラムだな」と推理して、勝手に正しい方法で実行してくれます。
