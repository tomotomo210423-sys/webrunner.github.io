パーフェクト使い方ガイド 📖

Web File Runner って何？🚀

プログラミングで作った「ゲーム」や「Webサイト」のファイルを、
インターネット上の特別なサーバーを用意しなくても、ブラウザの中で直接動かせる魔法の実験室だよ！

🧩 3つのファイルの役割（Webアプリの仕組み）

Webサイトやゲームは、主に**3種類のファイル**が協力して作られています。「家を建てること」に例えて説明するね！

📄 1. HTML（エイチティーエムエル）＝「家の骨組み」

画面に「ボタン」や「文字」をどこに置くかを決める設計図だよ。
（よくあるファイル名： index.html, game.html）

🎨 2. CSS（シーエスエス）＝「ペンキやインテリア」

ボタンを赤色にしたり、文字を大きくしたり、見た目をオシャレにデザインするためのファイルだよ。
（よくあるファイル名： style.css, main.css）

⚙️ 3. JavaScript（ジャバスクリプト）＝「電気やモーター」

「ボタンを押したらジャンプする」「敵が動く」「点数を計算する」といった、裏側の仕組みや"動き"を作るプログラムだよ。
（よくあるファイル名： script.js, app.js）

🎮 ツールの使い方ステップ

1
ファイルを開く 画面左上の「ファイルを開く」ボタンを押して、動かしたいプログラムのファイルを選ぼう。
バラバラになっている HTML、CSS、JavaScript をまとめて全部いっぺんに選ぶのがコツ！ツールが自動で1つに合体させてくれます。
2
コードをいじる（エディタ） 読み込んだファイルは左側の「エディタ」に文字として表示されるよ。
数字や色を書き換えて、自分だけのオリジナルゲームに改造してみよう！
3
コードを実行する 画面中央の「コードを実行」ボタンを押すと、右側のプレビュー画面にゲームやWebサイトが表示されて実際に遊べるようになるよ！
右上の「自動実行」のスイッチをONにすれば、コードを書き換えるたびに勝手に画面が更新されて超便利！

🖼️ 画像や音楽を使いたい時はどうするの？

ゲームのキャラクター画像（.png）や、BGM（.mp3）もこのツールで使えます！

  • ステップ1: コードのファイルと一緒に、画像や音楽のファイルも「ファイルを開く」で一緒に選んで読み込みます。
  • ステップ2: プログラム（HTMLやJSなど）の中に、hero.png のようにファイルの名前をそのまま書きます。
  • 完成！: 「コードを実行」を押すと、ツールが自動的に画像をコードの中にはめ込んで表示してくれます！
