パーフェクト使い方ガイド 📖 (Ver.1.2)

Web File Runner へようこそ！🎉

このツールは、あなたのスマホやPCに入っているプログラムのファイルを、
インターネット上のサーバーを使わずに、ブラウザだけで安全に動かせる魔法のツールです✨

基本の使い方と「魔法の合体機能」🧩

「ファイルを開く」ボタンからファイルを選んで、「コードを実行」を押すだけでOK！
さらに、複数のファイルを同時に選ぶと、ツールがファイルの種類を自動で推理し、一つのWebアプリとして合体させてから動かしてくれます。

ファイルの役割（Webアプリの仕組み）

  • 📄 .html (骨組み): 画面に表示される文字やボタンの配置を決めます。
  • 🎨 .css (装飾): 色を変えたり、オシャレなデザインに整えたりします。
  • ⚙️ .js (動作): ボタンを押した時の動きや裏側の仕組みを作ります。

よくある質問・困ったときは（FAQ）❓

🖼️ ゲームで使う画像やBGMは入れられるの？
はい、入れられます！
.png や .mp3 などの素材ファイルも一緒に選んでアップロードしてください。コードの中に <img src="hero.png"> のようにファイル名を書くだけで自動で結びつきます。
📄 「.txt」っていうただのメモ帳のファイルでも動く？
動きます！
ツールが中身のコードを読んで「あ、これはHTMLだな」「これはJavaScriptのプログラムだな」と推理して、勝手に正しい方法で実行してくれます。
