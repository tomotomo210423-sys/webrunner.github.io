Web開発の裏側がわかる！学習ガイド 📖

ブラウザは「魔法のキャンバス」

このツールは、プログラミングで作られたゲームやサイトを、
サーバーを使わずにあなたのブラウザの中で直接組み立てて動かす「統合開発環境(IDE)」です。

1. Webを構成する「3つの言語」の本当の姿

Webアプリは「家づくり」に例えられますが、コンピュータの内部ではもう少し複雑な連携が行われています。

📄 HTML (構造とツリー)

画面にボタンや文字を配置する設計図です。ブラウザはこれを読み込むと、DOM（Document Object Model）というツリー状のデータ構造をメモリ上に構築します。

🎨 CSS (装飾とボックス)

HTMLの各要素にデザインを適用します。CSSの世界では、全ての要素が四角い箱として扱われます（ボックスモデル）。

⚙️ JavaScript (動的ロジック)

ゲームの動きや計算を担当する「頭脳」です。イベント（出来事）を監視し、DOMを書き換えたりCSSを操作したりして動きを生み出します。

2. なぜ「サーバーなし」で画像や音楽が動くのか？

通常、Webサイトで画像を表示するには、インターネット上のサーバーに通信する必要があります。しかし、このツールでは通信を一切行いません。

🔍 深掘り：Base64（ベースロクジュウヨン）という魔法の変換

本ツールに画像（.pngなど）やZIPファイルを読み込ませると、裏側で画像データを「Base64」という長い文字列に自動で変換します。コード内のファイル名をこの文字列にすり替えてから実行するため、サーバーと通信しなくても画像や音楽を再生できます。

3. 実践！プロの開発機能（IDE）を使いこなそう

📁 ZIPファイルの一括読み込み

「ファイルを開く」ボタンから .zipファイル をそのまま選択すると、ツールが中身を展開し全ての関係性を保ったまま実行準備を整えます。

📝 コピー＆ペースト機能

「📋 コピー」はエディタで選択中のコードをコピーします（未選択時は全体をコピー）。「📝 ペースト」はClipboard APIが使えない環境でも専用モーダルで安全に貼り付けできます。
